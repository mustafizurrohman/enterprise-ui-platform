# Datepicker

The `DatepickerComponent` is a robust and accessible date and time selection
component designed for enterprise applications.

It provides a consistent user interface for selecting dates and times, with
support for localization, keyboard navigation, and intelligent manual input.

## Features

- **Automatic date and time detection:** Automatically switches between
date-only and date-time modes based on the configured `dateFormat`.
- **Intelligent manual input:** Supports manual entry with autocomplete,
German `Uhr` notation, and Unix timestamps.
- **Standalone component:** Integrates easily into modern Angular applications.
- **Angular Forms integration:** Supports template-driven and reactive forms
through `ControlValueAccessor` and `Validator`.
- **Accessibility:** Provides ARIA attributes and comprehensive keyboard
navigation.
- **Customizable:** Supports configurable labels, date formats, locales, and
disabled states.
