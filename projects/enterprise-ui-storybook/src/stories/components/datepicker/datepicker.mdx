# Datepicker

The `DatepickerComponent` is a robust and accessible date/time selection component designed for enterprise applications. It provides a consistent user interface for selecting dates and times, with support for localization, keyboard navigation, and manual input with autocomplete.

## Features

- **Automatic Date & Time Detection**: Automatically switches between date-only and date-time modes based on the provided `dateFormat`.
- **Intelligent Manual Input**: Supports manual entry with smart autocomplete, including support for German "Uhr" notation and Unix timestamps.
- **Standalone Component**: Easy to integrate in modern Angular applications.
- **Angular Forms Integration**: Full support for Template-driven and Reactive Forms (implements `ControlValueAccessor` and `Validator`).
- **Accessibility**: ARIA labels, roles, and full keyboard navigation support.
- **Customizable**: Control labels, display formats, locale, and disabled states.
